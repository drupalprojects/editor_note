
-- ABOUT --

Project for Drupal 7.
The modules improves Administration/Editorial Usability and provides
configurable "Editor Notes" field, which can be added to any Drupal entity.

-- OPTIONS --

* As an editor I can write status notes to other editors working on the same material.
* Note may be edited and deleted by its author or anyone with "Administer any editor note" permission.
* CRUD operations support Ajax and work without page reload.
